## Sales Analysis & Forecasting Project

### Problem
Analyze sales data to identify trends, profitability issues, and forecast future sales.

### Solution
- Cleaned and analyzed sales data
- Visualized key trends
- Forecasted future sales using time-series analysis
- Provided business recommendations

### Key Insights
- Technology category generated the highest sales
- High discounts negatively impact profit
- Some regions show negative profit despite strong sales

### Tools
- Python
- Pandas
- Matplotlib
- Seaborn

### Outcome
Clear insights and forecasts to support better business decisions.
